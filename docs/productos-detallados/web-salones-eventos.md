# Web para Salones de Eventos

Documento interno de venta y claridad comercial.

Objetivo: explicar este producto como una herramienta de reservación y presentación comercial, no solo como un sitio bonito, detallando con precisión qué cubre cada plan y qué no.

## 1. Qué es este producto

Web para Salones de Eventos es la solución de Devifly para recintos, jardines, terrazas, quintas y salones que necesitan mostrar mejor su espacio, responder dudas frecuentes y captar cotizaciones o reservas con más contexto comercial.

## 2. Qué significa la categoría: Eventos

En este producto, Eventos se refiere a soluciones para negocios que venden experiencias, fechas y espacios. Aquí el foco no está solo en mostrar imágenes, sino en ayudar a reservar mejor, cotizar mejor y filtrar prospectos con más intención real.

## 3. Qué problema resuelve

- El equipo responde lo mismo muchas veces por WhatsApp.
- Se pierden leads por respuesta tardía.
- El recinto no luce profesional en internet.
- No hay claridad sobre servicios, paquetes o disponibilidad.
- Todo el proceso comercial depende de llamadas o mensajes desordenados.

## 4. Cómo ayuda a generar más reservas

- Muestra el lugar de forma más atractiva.
- Ayuda a que el cliente entienda si el lugar le conviene.
- Facilita cotizaciones mejor filtradas.
- Reduce preguntas repetitivas.
- Mejora confianza antes del primer contacto.

## 5. Qué cubre la solución base del producto

| Característica | A qué se refiere | Qué resuelve |
| --- | --- | --- |
| Galería para vender la experiencia | Imágenes y contenido visual del recinto | Ayuda a que el cliente imagine su evento en el espacio |
| Formulario o cotizador | Mecanismo para captar interés comercial | Permite recibir leads con más contexto |
| Integración con WhatsApp | Canal directo para continuar conversación | Facilita el contacto inmediato |
| Enfoque en reservas | La estructura está pensada para vender fechas, paquetes o visitas | Evita que el sitio quede solo como folleto estático |

## 6. Planes

## 6.1 Esencial

### Qué resuelve

Resuelve la necesidad de tener una presencia web profesional para mostrar el salón y facilitar contacto sin todavía entrar a gestión comercial más avanzada.

### Qué cubre

- Presentación profesional del recinto.
- Galería visual base.
- Formulario de contacto.
- WhatsApp y ubicación.
- SEO local básico.

### Qué no cubre

- Cotizador en línea.
- Calendario de disponibilidad.
- CRM.
- Pagos o contratos digitales.
- Multi-sucursal.

| Diseño responsivo premium | Sitio bien adaptado a celular y desktop, con imagen cuidada | Mejora percepción profesional del salón |
| Galería de fotos (hasta 20) | Espacio para mostrar instalaciones, montaje y ambiente | Ayuda a vender visualmente la experiencia |

### Cuándo conviene venderlo

- Cuando quiere empezar a mostrarse mejor.
- Cuando el foco inicial es contacto, no todavía automatización.

## 8. Dominio y publicación

| Plan | Tipo de dominio | Ejemplo | Costo |
| --- | --- | --- | --- |
| Esencial | Subdominio Devifly | `tusalondeventos.devifly.dev` | Incluido sin costo adicional |
| Pro | Subdominio Devifly | `tusalondeventos.devifly.dev` | Incluido sin costo adicional |
| Premium VIP | Dominio propio (.com o .mx) | `tusalondeventos.com` | Primer año incluido en el precio del plan |

### Reglas de dominio

- Los planes Esencial y Pro se publican bajo un subdominio de devifly.dev. No hay costo de dominio.
- Premium VIP incluye el registro del dominio propio (.com o .mx) durante el primer año.
- A partir del año 2, la renovación es responsabilidad del cliente. Se puede gestionar a través del servicio Hosting + Dominio ($199 MXN/mes) o de forma independiente.
- El costo de renovación varía según extensión:
  - `.com`: ~$200–$360 MXN/año
  - `.mx`: ~$400–$800 MXN/año
  - Dominios premium o especiales: hasta $1,000+ MXN/año
- Si el cliente ya tiene dominio propio, se configura sin costo adicional de registro; solo se realiza la conexión técnica.
- Cualquier plan puede agregar dominio propio como upgrade pagando el costo de registro del dominio según extensión.
