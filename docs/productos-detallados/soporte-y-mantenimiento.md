# Soporte y Mantenimiento

Documento interno de venta y claridad comercial.

Objetivo: explicar con precisión qué resuelve cada paquete de soporte, qué cubre, qué no cubre y cuándo conviene recomendarlo para evitar malos entendidos después de la entrega.

## 1. Qué es esta línea de servicio

Después de entregar un sitio o solución, el cliente puede necesitar tres cosas distintas:

- que el proyecto siga publicado y seguro,
- que alguien haga cambios cuando se necesiten,
- que exista respaldo y seguimiento técnico.

Por eso Devifly separa post-entrega en dos niveles:

- Hosting + Dominio, que es la base operativa obligatoria para mantener el proyecto activo.
- Mantenimiento, que es opcional y depende de qué tanto acompañamiento quiera el cliente.

## 2. Diferencia entre hosting y mantenimiento

| Servicio | Qué significa | Qué resuelve |
| --- | --- | --- |
| Hosting + Dominio | Mantener el proyecto publicado, con dominio y SSL | Evita que el sitio quede fuera de línea o sin infraestructura base |
| Mantenimiento | Servicio de cambios, soporte y seguimiento técnico | Evita que el cliente se quede sin ayuda cuando quiera actualizar o mejorar |

## 3. Paquetes

## 3.1 Hosting + Dominio

### Qué resuelve

Resuelve la necesidad mínima de tener el proyecto activo, accesible y seguro. Sin este servicio, el sitio no tiene dónde vivir ni cómo sostener su dominio y certificado base.

### Qué cubre

- Hosting en servidor optimizado.
- Dominio personalizado como .com o .mx según contratación.
- SSL incluido para navegación segura.
- Renovación automática operativa según esquema contratado.

### Qué no cubre

- Cambios de contenido.
- Cambios de diseño.
- Nuevas secciones.
- Correcciones o evolución funcional fuera de incidencias de infraestructura.
- Gestión de campañas, SEO mensual o marketing.

### Detalle de características

| Característica | A qué se refiere | Qué resuelve |
| --- | --- | --- |
| Servidor optimizado | Infraestructura donde vive el sitio o solución | Mantiene disponibilidad y desempeño base |
| Dominio personalizado | Dirección web de la marca | Refuerza profesionalismo y facilita acceso |
| SSL incluido (https) | Certificado de seguridad para navegación protegida | Da confianza y evita alertas de sitio inseguro |
| Renovación automática | Continuidad operativa del servicio contratado | Reduce riesgo de caídas por olvido de pago o vencimiento |

### Desglose de costos de dominio

El servicio de Hosting + Dominio cubre la renovación anual del dominio. Sin embargo, el costo varía dependiendo de la extensión contratada:

| Extensión | Rango de costo anual | Notas |
| --- | --- | --- |
| `.com` | ~$200–$360 MXN | La más común y económica |
| `.com.mx` | ~$300–$500 MXN | Opción intermedia con enfoque México |
| `.mx` | ~$400–$800 MXN | Extensión premium para mercado mexicano |
| Dominios premium o especiales | Hasta $1,000+ MXN | Palabras clave muy buscadas o extensiones especiales |

**Importante:**
- Si el proyecto usa subdominio devifly.dev (planes básicos de producto), el costo de dominio no aplica y el servicio se limita a hosting + SSL.
- Si el cliente ya tiene dominio propio y solo necesita hosting, el costo de dominio no se incluye; solo se gestiona la conexión técnica sin cargo adicional.
- El costo del servicio Hosting + Dominio ($199 MXN/mes) cubre tanto infraestructura como renovación del dominio contratado. Si la extensión del dominio tiene un costo anual más alto (ej. .mx premium), puede aplicar un ajuste.
- El primer año de dominio puede estar incluido en planes Premium/Enterprise/VIP de los productos. A partir del año 2, el cliente asume la renovación directamente o a través de este servicio.

### Cuándo conviene recomendarlo

Siempre. Es el mínimo necesario para que el proyecto siga activo.

## 3.2 Mantenimiento Básico

### Qué resuelve

Resuelve escenarios donde el cliente necesita ayuda ocasional, pocos cambios al mes y una capa mínima de acompañamiento para no quedarse sin soporte.

### Qué cubre

- Hasta 2 cambios de contenido al mes.
- Soporte por WhatsApp con ventana aproximada de 48 horas.
- Monitoreo de disponibilidad.
- Hosting incluido dentro del paquete.

### Qué no cubre

- Rediseños completos.
- Cambios ilimitados.
- Nuevas funcionalidades complejas.
- Integraciones nuevas.
- Desarrollo de módulos o secciones grandes.
- Soporte inmediato o prioritario.

### Detalle de características

| Característica | A qué se refiere | Qué resuelve |
| --- | --- | --- |
| Hasta 2 cambios de contenido al mes | Ajustes puntuales en textos, imágenes o bloques simples | Da flexibilidad para mantener el sitio actualizado sin contratar desarrollo cada vez |
| Soporte por WhatsApp (48 hrs) | Canal directo con tiempo de respuesta estándar | Ofrece acompañamiento sin necesidad de un esquema premium |
| Monitoreo de disponibilidad | Revisión de que el sitio siga arriba | Reduce riesgo de que una caída pase desapercibida |
| Hosting incluido | El costo base de publicación ya viene dentro del plan | Simplifica operación administrativa para el cliente |

### Cuándo conviene recomendarlo

- Cuando el sitio cambia poco.
- Cuando el cliente solo quiere respaldo ocasional.
- Cuando el negocio todavía no necesita respuesta rápida.

## 3.3 Mantenimiento Pro

### Qué resuelve

Resuelve la necesidad de negocios más activos que necesitan cambios frecuentes, mejor velocidad de respuesta y algo más de seguimiento técnico y operativo.

### Qué cubre

- Hasta 5 cambios al mes de contenido y diseño ligero.
- Soporte prioritario con ventana aproximada de 24 horas.
- Reporte mensual de rendimiento.
- Backups semanales.
- Hosting incluido.

### Qué no cubre

- Cambios ilimitados.
- Nuevos módulos grandes.
- Replanteamiento completo de UX o branding.
- Integraciones complejas o desarrollo custom nuevo.
- Operación de marketing continua.

### Detalle de características

| Característica | A qué se refiere | Qué resuelve |
| --- | --- | --- |
| Hasta 5 cambios al mes (contenido + diseño) | Ajustes más frecuentes y con algo más de profundidad visual | Permite que el sitio se mantenga vivo sin entrar a rediseños mayores |
| Soporte prioritario (24 hrs) | Respuesta más rápida que en Básico | Disminuye tiempos muertos cuando el sitio es más importante para la operación |
| Reporte mensual de rendimiento | Resumen de estado o desempeño técnico básico | Da visibilidad sobre cómo va el proyecto tras la entrega |
| Backups semanales | Respaldo periódico de información o estado del proyecto | Reduce riesgo ante errores o incidentes |
| Hosting incluido | Infraestructura cubierta en el paquete | Facilita administración del servicio |

### Cuándo conviene recomendarlo

- Cuando el cliente actualiza seguido.
- Cuando el sitio participa activamente en ventas o captación.
- Cuando el negocio valora atención más rápida.

## 3.4 Mantenimiento VIP

### Qué resuelve

Resuelve la necesidad de negocios que quieren acompañamiento continuo, rapidez, cambios frecuentes y evolución más activa del proyecto sin abrir una cotización nueva por cada ajuste menor.

### Qué cubre

- Cambios ilimitados de contenido y diseño dentro de la lógica del proyecto.
- Soporte el mismo día en horario laboral.
- Nuevas secciones o funcionalidades incluidas dentro de un marco razonable de evolución.
- Reportes y recomendaciones mensuales.
- Hosting premium incluido.

### Qué no cubre

- Desarrollo ilimitado de módulos enterprise.
- Replantear por completo el producto cada mes.
- Proyectos paralelos nuevos no relacionados con la solución existente.
- Integraciones complejas o sistemas nuevos fuera del marco de mantenimiento.
- Soporte 24/7 si no se pacta expresamente.

### Detalle de características

| Característica | A qué se refiere | Qué resuelve |
| --- | --- | --- |
| Cambios ilimitados de contenido y diseño | Ajustes frecuentes dentro del mismo proyecto sin tope mensual típico | Evita fricción para marcas que cambian mucho |
| Soporte mismo día (horario laboral) | Atención más rápida en horario definido | Da tranquilidad a negocios que dependen del sitio más activamente |
| Nuevas secciones o funcionalidades incluidas | Espacio para evolución continua razonable del proyecto | Permite mejorar el sitio sin abrir mini-cotizaciones por cada ajuste menor |
| Reportes + recomendaciones mensuales | Seguimiento con sugerencias de mejora | Convierte mantenimiento en evolución y no solo en corrección |
| Hosting premium incluido | Infraestructura con nivel más alto dentro del esquema del servicio | Acompaña necesidades de clientes más activos |

### Cuándo conviene recomendarlo

- Cuando el sitio cambia mucho.
- Cuando el cliente quiere rapidez y acompañamiento cercano.
- Cuando el proyecto sigue creciendo después del lanzamiento.

**Flujo post año 1:** Cuando un plan Premium/Enterprise/VIP incluyó dominio propio el primer año, a partir del año 2 el cliente debe:
1. Contratar el servicio de Hosting + Dominio ($199 MXN/mes) que ya incluye la renovación, o
2. Renovar el dominio por su cuenta y solo contratar hosting si lo necesita.

## 6. Cómo explicarlo en una reunión

"Hosting mantiene el proyecto vivo. Mantenimiento lo mantiene útil, actualizado y acompañado. Básico es para cambios ocasionales, Pro para negocios activos y VIP para acompañamiento continuo y evolución constante."

## 7. Condiciones comerciales del servicio post-entrega

| Concepto | Detalle |
| --- | --- |
| Cobro | Mensual, con cargo recurrente o pago anticipado según acuerdo |
| Cancelación | Se puede cancelar con aviso de 15 días antes del siguiente ciclo |
| Compromiso mínimo | Sin permanencia mínima obligatoria (recomendado mínimo 3 meses) |
| Cambios fuera de alcance | Se cotizan aparte como proyecto adicional |
| Vigencia de precios | Los precios aplican al momento de contratación y pueden ajustarse con aviso previo |

## 8. Reglas de claridad comercial

- Hosting no es mantenimiento.
- Mantenimiento no es rediseño total ilimitado.
- Todo cambio fuera del marco del paquete puede cotizarse aparte.
- Funcionalidades nuevas grandes, integraciones complejas o cambios de alcance mayor deben tratarse como proyecto adicional o evolución especial.
