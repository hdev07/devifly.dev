# Devifly 🚀

¡Bienvenido al repositorio oficial del sitio web y portafolio de **Devifly**!

Devifly es una agencia especializada en desarrollo web moderno, creando experiencias digitales excepcionales. Nuestro catálogo de servicios abarca desde Landing Pages de alto impacto e Invitaciones Digitales interactivas, hasta el desarrollo de complejas plataformas SaaS (como MoviBase y Confecdotario).

## 🛠 Stack Tecnológico

Este proyecto ha sido desarrollado utilizando las tecnologías más modernas y eficientes del ecosistema frontend:

- **Framework:** [Vue 3](https://vuejs.org/) (Composition API & `<script setup>`)
- **Build Tool:** [Vite](https://vitejs.dev/) - Entorno de desarrollo ultra rápido
- **Estilos:** [Tailwind CSS v4](https://tailwindcss.com/) - Framework CSS de utilidad
- **Routing:** [Vue Router 4](https://router.vuejs.org/) - Enrutamiento dinámico
- **Iconografía:** [Lucide Vue Next](https://lucide.dev/) - Iconos hermosos y consistentes
- **Internacionalización:** [Vue I18n](https://kazupon.github.io/vue-i18n/) - Soporte multilenguaje
- **Utilidades:** [VueUse](https://vueuse.org/) - Colección de utilidades Composition API

## 📂 Estructura Principal del Proyecto

- `src/views/`: Vistas principales y páginas de la aplicación (Home, Servicios, etc.)
- `src/views/demos/`: Demos interactivas de nuestros servicios (por ejemplo, Invitaciones Básica, Pro y Premium).
- `src/components/`: Componentes reutilizables de UI.
- `src/data/`: Datos estáticos y configuración del catálogo de productos (`products.js`).
- `src/assets/`: Recursos estáticos como imágenes, videos y hojas de estilo globales.

## 🚀 Configuración y Ejecución Local

Para correr este proyecto en tu entorno local, sigue estos pasos:

### 1. Clonar el repositorio y acceder a la carpeta:
```bash
cd devifly.dev
```

### 2. Instalar dependencias:
Asegúrate de usar `npm`, `yarn`, `pnpm` o `bun`.
```bash
npm install
```

### 3. Iniciar el servidor de desarrollo:
```bash
npm run dev
```
El proyecto estará disponible localmente, generalmente en `http://localhost:5173`.

### 4. Construcción para Producción:
```bash
npm run build
```
Esto generará los archivos estáticos optimizados en la carpeta `dist`.

### 5. Previsualizar la construcción:
```bash
npm run preview
```
