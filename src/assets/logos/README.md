# Logos de clientes

Suelta aqui los SVGs de logos de clientes (formato `.svg`).

Se cargan automaticamente en el componente `TrustBar` mediante `import.meta.glob`.

Recomendaciones:
- Color `currentColor` o monocromaticos (se renderizan en escala de grises sobre el fondo oscuro).
- Altura visual ~32-40px.
- Nombres descriptivos: `movibase.svg`, `confecdotario.svg`, `musion.svg`, etc.

Si no hay logos en este folder, el TrustBar mostrara nombres de marcas con tipografia displays (fallback automatico).
